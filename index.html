Html semantico y sus elementos

los elementos semanticos, no renderizan absolutamente nada

"aside es una barra lateral del menu"
"footer es para el pie de pagina del sitio web"
"header se usa para el encabezado de nuestro sitio web"
"main contenido principal"
"nav implementa menu de navegacion"
"section se usa para las diferentes secciones del sitio"
" time se usa para el tiempo"
"article, me esta indicando un articulo de mi pagina"

Elemento div

al igual que los elementos semanticos, no renderiza absolutamente nada, pero nos ayuda a maquetar el contenido de nuestra pagina o sitio.

El elemento div es como una caja, donde puedo agregar, parrafos imagenes y todo lo que yo quiera, luego con css lo puedo formatear como yo quiera
